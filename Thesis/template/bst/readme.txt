agrosy.bst 		- Der alte AgRoSy-Style. Es wurden einige Fehler behoben, so dass er mitlerweile bei allen Einträgen problemlos kompilieren sollte.

agrosy_deutsch.bst 	- Zum Erstellen von Literaturverzeichnissen auf deutsch. Ein Style, der die bekannten AgRoSy-Bibitems benutzt, sich aber ansonsten an dem Dinat-Style orientiert. 
                          Dieser entspricht der DIN 1505 zum Erstellen von Literaturverzeichnissen.

agrosy_deutsch_nummern.bst	- Wie oben, nur mit Nummer als Items.

agrosy_lni.bst 		- Ein weiterer Style zum erstellen von Literaturverzeichnissen auf deutsch. Er benutzt die bekannten AgRoSy-Bibitems und entspricht sonst dem lni-Style den der 
                          Springer-Verlag verwendet.

agrosy_lni_nummern.bst 	- Wie oben, nur mit Nummer als Items.

agrosy_english.bst 	- Estellt Literaturverzeichnisse auf englisch. Der Style benutzt die bekannten AgRoSy-Bibitems und entspricht sonst dem IEEE-Style.

agrosy_english_numbers.bst	- Wie oben, nur mit Nummer als Items.


Abhängig davon mit welcher Sprache agrosy.cls eingebunden wird, wird standardmäßig entweder agrosy_deutsch.bst oder agrosy_english.bst ausgewählt. Wenn man einen anderen Style benutzen will,
muss man dies von Hand einstellen.
